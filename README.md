# Privacy Policy — Patho Viewer Spacemouse

**Last updated: March 2026**

## Data Collection

Patho Viewer Spacemouse does **not** collect, store, or transmit any personal data.

## Clipboard Access

The extension reads clipboard content solely to detect medical viewer URLs 
(containing `/viewer/`) and automatically open them in the browser. 
Clipboard content is never stored, logged, or sent to any server.

## Settings Storage

User preferences (SpaceMouse speed, keyboard shortcuts, UI options) are 
stored locally in the browser using `chrome.storage.sync`. 
This data never leaves the browser.

## Third Parties

This extension does not share any data with third parties.

## Contact

For any questions, contact the extension developer.
